Tienda – Aplicación E-commerce en Django

Proyecto desarrollado en Django. Incluye gestión de productos, carrito de compras, autenticación de usuarios y una API REST para el acceso a los productos. La aplicación está organizada en módulos independientes y cuenta con un sistema de plantillas y archivos estáticos listo para producción.

Características principales:

Listado y detalle de productos

Carrito de compras funcional

Autenticación y gestión de usuarios

API REST con Django REST Framework

Arquitectura modular y extensible
